    * Day of week
    * Time of day
    * Guest count
    * Event duration
    * Package chosen
    * Type of event
  * One-hour weddings start at $1,000; 2-hour weddings start at $5,000; evening and weekend rentals range $10,000–30,000.

  * **$83/event and up**
  * Ceremony fee varies depending on: 
    * Day of week
    * Time of day
    * Guest count
    * Event duration
    * Package chosen
    * Type of event
  * Civil ceremonies start at $83. Private ceremonies start at $1,000 and can include up to 100 guests.
