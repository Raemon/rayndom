# Corporate Seminar

Back in October, Kohl Mansion partnered with La Bocca Fina for a corporate event that was held. A seminar was held throughout the day for the employees ending with a nice buffet style dinner and a fun array of desserts. The Great Hall was the main space used along with the Library which was set up with TV screens, monitors and chairs for guests to view easily. 

A BIG thank you to:  
-Catering: La Bocca Fina  
-Beverages: Smith Beverages  
-Music/Lighting/AV: Verducci Events
